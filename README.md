# TOBI x SHADOW OTP - Bot Telegram

🤖 Bot de vente de numéros virtuels OTP

## Fonctionnalités
- Achat de numéros virtuels (WhatsApp, Telegram, TikTok, etc.)
- Dépôts via Mobile Money
- Historique des commandes
- Panel Admin
- Système de parrainage

## Installation

1. Clone le repo
2. `npm install`
3. Copie `.env.example` vers `.env` et remplis les variables
4. `npm start`

## Déploiement sur panel
- Node.js >= 18
- SQLite
- Port 3000
